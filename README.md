# NSNumber

## 概要
汎用的な数値のラッパクラスです。  
Int, Float, Double, Bool型へ変換可能で、as演算子によるアップキャストが可能となっています。

## 関連クラス
NSValue

## フレームワーク
Foundation.framework

## サポートOSバージョン
iOS2.0以上

## 参考
https://developer.apple.com/reference/foundation/nsnumber
